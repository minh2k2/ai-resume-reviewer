## AI Resume Reviewer

A frontend demo application that simulates an AI-powered resume analysis tool.

### Features
- Upload PDF resume
- Mock AI analysis with realistic delay
- Resume score with progress visualization
- Strengths / Weaknesses / Suggestions sections
- Clean Material UI layout

### Tech Stack
- React + Vite
- Material UI
- Mock AI response (ready for backend integration)

### Why this project
Built to demonstrate:
- Component-based architecture
- UX-focused AI interfaces
- Clean and readable frontend code
